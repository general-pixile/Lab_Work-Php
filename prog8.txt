4. To verify whether a function exists before calling it.
Create a function named welcomeStudent().
Use function_exists() to check its existence.
If the function exists, call it.
Otherwise display an error message.

// Function to welcome student
function welcomeStudent()
{
    echo "<h3>Welcome Student!</h3>";
}

// Calling collegeInfo()
collegeInfo();

// Calling studentDetails()
studentDetails("Rahul Sharma", "BCA2025001", 3);

// Calling calculatePercentage()
$percentage = calculatePercentage(85, 78, 92, 88, 80);
echo "Percentage: " . number_format($percentage, 2) . "%<br><br>";

// Checking if welcomeStudent() exists before calling it
if (function_exists("welcomeStudent")) {
    welcomeStudent();
} else {
    echo "Error: Function welcomeStudent() does not exist.";
}
